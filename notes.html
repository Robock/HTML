What is HTML?

HTML is short for Hypertext Markup Language. It is a file system by which individuals communicate with computers. We write code in HTML format. Just like we write text in Microsoft Word to communicate with other humans, we use Text Editors to communicate with computers. A Text Editor is a piece of software that is used to write, edit and save code. We have a few options of text editors that we can use like Atom, Sublime Text, Brackets, etc.

In order to write code, we need to think like a programmer. The 5 ways programmers think are:

Procedural Thinking
A method of thinking that involves several steps.
Abstract Thinking
Finding similarities in different things to create Generality.
Systematic Debugging
Finding possible causes of problems and fixing it.
Technological Empathy
A computer is a tool that outputs information based on what a user inputs and programmers need to be mindful of that.
Systems Thinking
Breaking big problems into smaller pieces.

The World Wide Web

The major components of the web are 1) Browser, 2) Internet, 3) HTTP and 4) Servers. A browser is something like an internet explorer, chrome, firefox or safari that makes request via the internet to servers. These requests are made using an internet connection and a protocol called HTTP. HTTP stands for Hypertext Transfer Protocol. Servers are dedicated computers that respond to the request from users with files that is displayed in the users' browser.
